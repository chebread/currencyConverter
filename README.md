# 환율 계산 프로그램 📊

### 만든 목적
환율 계산을 일일히 검색하기 귀찮아서 만들었습니다 🤣

### 용도
용도는 아무렇게나 사용하시면 되요 🙂

### 사용 🔨
사용은 git clone 하셔서 /usr/bin/python3 (클론된 디렉터리 주소) 로 빌드해서 사용하세요 🤩
